Capstone

Started: 10/02/2024, Latest Commit:

For my capstone project, my team and I developed a Student Attendance Tracking System for the Hynes Institute at Iona University. The system is designed to streamline and digitize the process of tracking student attendance at events hosted by the institute. The project involved building a user-friendly web application that allows students to easily sign in to events using either their ID number or a QR code. The system is integrated with a secure backend database to store attendance data, which staff can efficiently query, analyze, and export for reporting purposes. The project not only focused on the technical development of the app, using React and Node.js, but also required extensive collaboration with various university departments to ensure data privacy compliance under FERPA and other regulations. The system provides a seamless and consistent user experience, using Iona University's branding, while improving data accuracy and efficiency for event management.

Avocado Content Creator

22 February, 2024

This project focuses on building an automated TikTok news content account targeting a demographic of 18-40-year-olds. The primary goal is to deliver major news events in a straightforward, informational style while also covering lesser-known stories for diverse content. Using Python scripts and AI tools like GPT for text generation and DALL-E for image creation, the project aims to create content four times daily. Additionally, it involves leveraging TikTok's API for automation, maintaining AI models, and ensuring compliance with TikTok policies. A manual review process will ensure the quality of AI-generated content, and the entire project journey is documented for continuous updates and improvements.
